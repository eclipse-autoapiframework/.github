# Welcome to the eclipse-autoapiframework GitHub organization
This space is where the project artifacts of the [Eclipse Automotive API
Framework](https://projects.eclipse.org/projects/automotive.autoapiframework) project get developed
and maintained.

## Repository overview
This organization includes the following repositories:
| Repository  | Description |
| ----------- | ----------- |
| [.github](https://github.com/eclipse-autoapiframework/.github) | Organization README and project logo files. |
| [.eclipsefdn](https://github.com/eclipse-autoapiframework/.eclipsefdn) | Configuration files related to the Eclipse Foundation. |
| [autoapiframework](https://github.com/eclipse-autoapiframework/autoapiframework) | Examples, on the one hand, that illustrate the combined use of contributions to this project. On the other hand, blueprints that show the integration with other open source projects in the context of Software-Defined Vehicles (SDV). |
| [application framework](https://github.com/eclipse-autoapiframework/application-framework) | Framework that addresses the middleware-agnostic design, development, and test of distributed applications for microprocessor-based systems. |
| [vss-gui-tool](https://github.com/eclipse-autoapiframework/vss-gui-tool) | Tool for the customization and editing of the [Vehicle Signal Specification (VSS)](https://covesa.github.io/vehicle_signal_specification/) catalog. |
